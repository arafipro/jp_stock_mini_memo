# 日本株ひとこと投資メモ

日本株ひとこと投資メモは<u>東証市場再編対応版</u>です。

「日本株投資ひとことメモ」は、日本の株式市場での投資を気軽に管理することができるツールです。\
ひとことで投資の意図やアイデアを記録できるので日本株投資をするにあたり手軽に利用できるのが魅力です。\
また日本株投資ひとことメモを使えば日々の投資をよりスムーズに管理できるようになります。
